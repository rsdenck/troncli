# CircleCI CLI

- **Repo:** CircleCI-Public/circleci-cli
- **Description:** CLI for CircleCI
- **Category:** CI/CD

## Commands

### circleci setup
Set up the CLI with your API token.

### circleci pipeline
Manage pipelines.

### circleci config
Check and process config files.

### circleci local execute
Run a job in a container on local machine.

## Install
```bash
curl -fLSs https://raw.githubusercontent.com/CircleCI-Public/circleci-cli/main/install.sh | bash
```
