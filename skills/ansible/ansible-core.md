# Ansible Core CLI

- **Repo:** https://github.com/rsdenck/skillnux/automation/ansible_pull.go
- **Description:** Automation and configuration management tool
- **Category:** Automation

## Commands

### ansible
Run tasks against hosts.

### ansible-playbook
Run Ansible playbooks.

### ansible-galaxy
Manage Ansible roles and collections.

### ansible-vault
Encrypt and decrypt sensitive data.

### ansible-navigator
Text-based user interface for Ansible.

## Install
```bash
sudo apt update
sudo apt install ansible
```
