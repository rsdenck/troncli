# Packaging

TronCLI provides official packages for major Linux distributions.

## Debian / Ubuntu (.deb)

The `.deb` package installs the binary to `/usr/bin/troncli` and the man page to `/usr/share/man/man1/troncli.1.gz`.

Dependencies: `libc6`

## RHEL / Fedora (.rpm)

The `.rpm` package follows the standard directory structure.

## Arch Linux (AUR)

TronCLI is available in the AUR as `troncli-bin` or `troncli-git`.

## Snap

Install via Snapcraft:

```bash
sudo snap install troncli
```
