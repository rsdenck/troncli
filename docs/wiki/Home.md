# Home

Welcome to the **TronCLI** Wiki!

TronCLI is a production-grade CLI and TUI for comprehensive Linux system administration. It is designed for DevOps, SREs, and SysAdmins who need reliable, idempotent, and verifiable system modifications.

## Core Features

- **Universal Package Management**: Support for apt, dnf, yum, pacman, apk, zypper.
- **Service Management**: Control systemd, openrc, and sysvinit services.
- **Process & Resource Monitoring**: Real-time TUI dashboard.
- **Network Configuration & Diagnostics**: Configure interfaces, scan ports, trace routes.
- **User & Group Management**: Add, modify, and delete users/groups.
- **Disk & Filesystem Operations**: Monitor usage, manage LVM.
- **Security Auditing**: Check for vulnerabilities, audit logs.
- **Container Management**: Docker and Podman integration.
- **Remote Execution**: SSH client with profile support.
