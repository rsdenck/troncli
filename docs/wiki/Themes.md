# Themes

TronCLI supports a theming engine with ANSI and TrueColor support.

## Available Themes

### Cyberpunk
A neon-inspired theme with high contrast.
- Primary: Neon Pink (#FF00FF)
- Secondary: Cyan (#00FFFF)
- Background: Black (#000000)

### Matrix
A classic hacker theme.
- Primary: Green (#00FF41)
- Secondary: Dark Green (#003B00)
- Background: Black (#000000)

## Switching Themes

You can switch themes via the CLI flag:

```bash
troncli --theme cyberpunk
troncli --theme matrix
```

Or configure it permanently in `config.yaml`.
