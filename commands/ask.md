# Ask#

Ask questions to AI providers (Ollama, OpenAI, Claude, NVIDIA Build)

## Usage#

```
nux ask <question>
```

## Description#

Sends a question to the configured AI provider and returns the response. Supports multiple providers with automatic fallback.

## Examples#

Ask about Linux commands:
nux ask "How to list files in Linux?"

Ask for system troubleshooting:
nux ask "Why is my Apache server not starting?"
